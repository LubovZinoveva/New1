Чтобы выделить текст в ненумерованный список, добавьте звездочку (*). Например:
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы выделить текст в нумерованный список, просто пронумеруйте пункты. Например:
1. Пункт 1 
2. Пункт 2
